# Backup-Usuario
Em um HD EXTERNO ou HD Secundario Crie uma pasta com o nome Backup e mova a pasta Script Backup CMD para ela.
Os dados do usuario seram copiados para esta pasta, com o nome
do computador.
A Letra da Unidade de ve ser D:

para Mudar a letra da unidade Acesse:

CRIAR E FORMATAR PARTIÇOES DE DISCO.
CLIQUE COM O BOTÃO DIREITO E VA EM = ALTERAR LETRA E CAMINHO DA UNIDADE, ALTERAR.
TROQUE A LETRA DA UNIDADE PARA D:.


ATENÇÃO: Caso ele não encontre a pasta Backup ele Criara a pasta.
